# Minimalistic Photography Portfolio Website

This is a simple, clean, and minimalistic photography portfolio website built with HTML, CSS, and JavaScript (no frameworks).

## Features
- Responsive gallery grid for photos
- About section
- Contact form (demo only)
- Minimal, modern design

## Structure
- `index.html` — Main page
- `css/main.css` — Stylesheet
- `js/main.js` — JavaScript for contact form
- `assets/` — Placeholder images (replace with your own photos)

## Usage
1. Replace the images in the `images/` folder with your own photos.
2. Open `index.html` in your browser to view the site.
